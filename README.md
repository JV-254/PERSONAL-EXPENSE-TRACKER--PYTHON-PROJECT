# PERSONAL-EXPENSE-TRACKER--PYTHON-PROJECT
## Project Overview

The Personal Expense Tracker is a data analysis project built using Python and Pandas. The project helps users track, organize, and analyze their daily expenses to better understand spending habits and manage personal finances.

## Features

* Load expense data from a CSV file
* Calculate total spending
* Calculate average expense amount
* Identify the highest and lowest expenses
* Analyze spending by category
* Filter expenses based on specific criteria
* Track spending by payment method
* Generate monthly spending summaries
* Calculate spending percentages for each category

## Technologies Used

* Python
* Pandas
* Jupyter Notebook

## Dataset Structure

| Column         | Description                |
| -------------- | -------------------------- |
| Date           | Date of transaction        |
| Category       | Expense category           |
| Description    | Description of the expense |
| Amount         | Amount spent (KSh)         |
| Payment_Method | Method of payment          |
